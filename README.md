# tasks-app

Uma lista de tarefas minimalista, estilosa e funcional, construída com **Electron**, focada em um design "cozy" e alta performance. O app possui um visual personalizado com tons de rosa, vermelho e verde, incluindo um calendário dinâmico e barra de progresso em tempo real.

## Como rodar o projeto localmente

Se você quiser explorar o código ou testar o app na sua máquina, siga os passos abaixo:

> **Nota:** Certifique-se de ter o [Node.js](https://nodejs.org/) instalado.

```bash
# 1. Clone este repositório para sua máquina
git clone [https://github.com/isabellalulu/tasks-app.git]

# 2. Entre na pasta do projeto
cd tasks-app

# 3. Instale as dependências (Electron e bibliotecas)
npm install

# 4. Execute o aplicativo
npm  start
